## Oracle 연계

### 개요

회귀 분석 진행 상황을 Oracle DB에 실시간 업데이트하기 위해 **python-oracledb**를 사용합니다.

```

│                                                                          │
│  ┌──────────────┐        ┌──────────────┐      ┌──────────────┐       │
│  │  PySpark     │        │   Sklearn    │      │  python-     │       │
│  │  (Spark)     │───────▶│  (ML Core)   │─────▶│   oracledb   │       │
│  │              │        │              │      │              │       │
```

### python-oracledb 구현

#### 함수 시그니처

```python
def update_oracle_status(spark, job_id, area, status, current_step, message, progress):
    """Update regression status in Oracle using python-oracledb"""
```

#### 파라미터

| 파라미터 | 타입 | 설명 | 예시 |
|---------|------|------|------|
| `spark` | SparkSession | Spark 세션 (현재 미사용) | `spark` |
| `job_id` | str | 작업 ID (PK) | `regression_data.csv` |
| `area` | str | 클러스터 영역 | `ich`, `wxh` |
| `status` | str | 상태 코드 | `STARTED`, `COMPLETED` |
| `current_step` | str | 현재 단계명 (현재 미사용) | `Data Loading` |
| `message` | str | 상태 메시지 (현재 미사용) | `Data loaded: 299 rows` |
| `progress` | int/float | 진행률 (0-100) | `0`, `30`, `100` |

#### 전체 코드

```python
def update_oracle_status(spark, job_id, area, status, current_step, message, progress):
    """Update regression status in Oracle using python-oracledb"""

    import oracledb
    import os

    # Oracle 연결 정보
    oracle_user = os.getenv("ORACLE_USER", "system")
    oracle_password = os.getenv("ORACLE_PASSWORD", "Oracle123")
    oracle_dsn = os.getenv("ORACLE_DSN", "localhost:1521/FREEPDB1")

    try:
        # 연결 생성
        connection = oracledb.connect(
            user=oracle_user,
            password=oracle_password,
            dsn=oracle_dsn
        )
        cursor = connection.cursor()

        # MERGE SQL (UPSERT) - jobid가 키
        merge_sql = """MERGE INTO regression_status t
USING (SELECT :1 jobid FROM dual) s
ON (t.jobid = s.jobid)
WHEN MATCHED THEN
    UPDATE SET status=:2, progress=:3, update_time=SYSTIMESTAMP
WHEN NOT MATCHED THEN
    INSERT (jobid, area, status, progress, start_time)
    VALUES (:4, :5, :6, :7, SYSTIMESTAMP)"""

        # 파라미터 바인딩 (SQL Injection 방지)
        cursor.execute(merge_sql, [
            job_id,           # :1 (MATCHED 조건)
            status,           # :2 (UPDATE status)
            progress,         # :3 (UPDATE progress)
            job_id,           # :4 (INSERT jobid)
            area,             # :5 (INSERT area)
            status,           # :6 (INSERT status)
            progress          # :7 (INSERT progress)
        ])

        connection.commit()
        print(f"[ORACLE] {status} - {progress}%")

    except Exception as e:
        print(f"[ERROR] Oracle update failed: {e}")
    finally:
        if 'cursor' in locals():
            cursor.close()
        if 'connection' in locals():
            connection.close()
```

### MERGE SQL 동작

#### UPSERT 로직

```sql
MERGE INTO regression_status t
USING (SELECT :1 jobid FROM dual) s
ON (t.jobid = s.jobid)
WHEN MATCHED THEN
    UPDATE SET status=:2, progress=:3, update_time=SYSTIMESTAMP
WHEN NOT MATCHED THEN
    INSERT (jobid, area, status, progress, start_time)
    VALUES (:4, :5, :6, :7, SYSTIMESTAMP)
```

#### 실행 동작

| 조건 | 동작 | SQL |
|------|------|-----|
| **jobid 존재** | UPDATE | `UPDATE SET status='...', progress=...` |
| **jobid 없음** | INSERT | `INSERT (jobid, area, status, progress, ...)` |

#### 파라미터 바인딩 순서

| 순서 | 바인드 변수 | 파라미터 | 설명 |
|------|------------|---------|------|
| 1 | :1 | job_id | MERGE 조건 (MATCHED 확인) |
| 2 | :2 | status | UPDATE 할 status |
| 3 | :3 | progress | UPDATE 할 progress |
| 4 | :4 | job_id | INSERT 할 jobid |
| 5 | :5 | area | INSERT 할 area |
| 6 | :6 | status | INSERT 할 status |
| 7 | :7 | progress | INSERT 할 progress |

### SQL Injection 방지

#### 파라미터 바인딩 사용

```python
# ✅ 안전한 방식 (파라미터 바인딩)
cursor.execute(merge_sql, [job_id, status, progress, ...])
```

#### 보안 비교

| 방식 | 구현 | SQL Injection 가능성 |
|------|------|---------------------|

| 파라미터 바인딩 | `cursor.execute(sql, [status])` | ✅ 안전 |

```python
def test_oracle_connection(spark):
    """Test Oracle JDBC connection (간단한 현재 시간 조회)"""
    try:
        ORACLE_JDBC_JAR = os.getenv("ORACLE_JDBC_JAR", "/root/oracle_jdbc/jdbc/ojdbc8.jar")
        if not os.path.exists(ORACLE_JDBC_JAR):
            print(f"[SKIP] Oracle JDBC JAR not found: {ORACLE_JDBC_JAR}")
            return False

        ORACLE_JDBC_URL = os.getenv("ORACLE_JDBC_URL", "jdbc:oracle:thin:@//localhost:1521/FREEPDB1")
        ORACLE_USER = os.getenv("ORACLE_USER", "system")
        ORACLE_PASSWORD = os.getenv("ORACLE_PASSWORD", "Oracle123")
        ORACLE_DRIVER = "oracle.jdbc.OracleDriver"

        print("[INFO] Testing Oracle connection...")

        # Query: Get current date/time from Oracle
        df_oracle = spark.read \
            .format("jdbc") \
            .option("url", ORACLE_JDBC_URL) \
            .option("dbtable", "(SELECT SYSDATE AS current_date, CURRENT_TIMESTAMP AS current_timestamp FROM dual)") \
            .option("user", ORACLE_USER) \
            .option("password", ORACLE_PASSWORD) \
            .option("driver", ORACLE_DRIVER) \
            .load()

        print("[SUCCESS] Oracle connection test passed!")
        print("Oracle Date/Time:")
        df_oracle.show(truncate=False)

        return True

    except Exception as e:
        print(f"[ERROR] Oracle connection failed: {e}")
        return False
```

## Apache Iceberg 구성

### 개요

Apache Iceberg는 **ACID 트랜잭션**, **시간 여행(Time Travel)**, **스키마 진화**를 지원하는 차세대 테이블 형식입니다.

### 아키텍처

```
┌─────────────────────────────────────────────────────────────────────────┐
│                    회귀 분석 파이프라인 (Iceberg)                        │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│  ┌──────────────┐        ┌──────────────┐      ┌──────────────┐       │
│  │  PySpark     │        │   Sklearn    │      │  Apache      │       │
│  │  (Spark)     │───────▶│  (ML Core)   │─────▶│  Iceberg     │       │
│  │              │        │              │      │              │       │
│  └──────────────┘        └──────────────┘      └──────┬───────┘       │
│                                                      │                │
│                                                      ▼                │
│                                          ┌──────────────────────┐     │
│                                          │  MinIO (S3 호환)      │     │
│                                          │  - s3a://ic-ias/     │     │
│                                          │  - Parquet 파일       │     │
│                                          │  - ACID 트랜잭션     │     │
│                                          └──────────────────────┘     │
│                                                                          │
└─────────────────────────────────────────────────────────────────────────┘
```

### Iceberg vs Hive 비교

| 항목 | Hive (이전) | Iceberg (현재) | 개선 |
|------|-------------|----------------|------|
| **테이블 형식** | Parquet + 파티셔닝 | Iceberg (ACID) | 트랜잭션 지원 |
| **저장소** | HDFS | MinIO S3 | S3 호환 |
| **카탈로그** | Hive Metastore | Iceberg Hadoop Catalog | 단순화 |

### Iceberg 파티셔닝

#### dt 파티션 (날짜 기반)

```python
def create_insert_value(*str_values):
    # Get current date for dt partition (format: YYYYMMDD)
    dt_partition = datetime.datetime.now().strftime("%Y%m%d")
    str_query = " ("
    for idx, value in enumerate(str_values):
        str_query += "'" + str(value) + "',"
    # Add dt partition value at the end
    str_query += "'" + dt_partition + "')"
    return str_query
```
